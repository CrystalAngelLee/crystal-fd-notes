# 数组的解构

```js
const arr = [100, 200, 300]
const [, , baz] = arr
console.log(baz)

const [foo, ...rest] = arr
console.log(rest)

const [foo, bar, baz = 123, more = 'default value'] = arr
console.log(bar, more)

// use
const [, rootdir] = path.split('/')
console.log(rootdir)
```



# 对象的解构

```js
const obj = { name: 'zce', age: 18 }

const name = 'tom'
// objName 为 name 的重命名
const { name: objName = 'jack' } = obj
console.log(objName)
```
