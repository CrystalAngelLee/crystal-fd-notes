### ECMA Script 新特性

我们通常将 ECMA Script 看作是JS的标准化规范，实际上JS是ES的扩展语言。ES只是提供了最基本的语法，并不能支持我们实际应用的功能开发。

浏览器中的JS = `ES + Web APIs(DOM + BOM)`

Node中的JS = `ES + Node APIs(fs + net + etc.)`

**JS 语言本身指的就是ES**
